# Character Images Directory

Place your local character images in this directory.

Recommended dimensions: 800x1200px (Portrait)
Format: JPG or WEBP for better performance.

## Mapping
- Elias: ./elias.jpg
- Victor: ./victor.jpg
- Isabel: ./isabel.jpg
- Bautista: ./bautista.jpg

Once images are added, update `constants.tsx` to point to these local paths instead of the Unsplash URLs.
